DataViz 低代码看板平台

企业级低代码可视化数据看板搭建平台。通过 JSON Schema 驱动 + 通用数据桥接架构，实现异构数据源的统一接入与 ECharts 动态渲染，配合 GridStack.js 拖拽构建器，让非技术人员也能快速搭建专业数据大屏。

项目概述

DataViz Dashboard 是一个面向企业 BI 场景的低代码数据可视化平台。核心设计理念是 Schema 驱动：所有看板配置（布局、组件、数据源绑定、图表类型）均以 JSON Schema 形式存储，前后端完全解耦。前端通过解析 Schema 动态渲染 ECharts 图表，后端通过通用数据桥接层归一化异构数据源（MySQL / PostgreSQL / SQLite / HTTP API）的查询结果。

核心功能

📊

Schema 驱动渲染

看板布局、组件、数据绑定全部通过 JSON Schema 描述，前端动态解析渲染，无需硬编码。

🔗

通用数据桥接

一行 SQL 或一个 URL 即可接入数据。支持 MySQL、PostgreSQL、SQLite、HTTP API 四种数据源类型，自动归一化为 ECharts 标准格式。

🎲

拖拽搭建器

基于 GridStack.js 的所见即所得画布：拖拽调整组件位置和尺寸、右侧面板配置数据源和图表参数、JSON 编辑器实时预览 Schema。

⏴

版本回滚

每次保存自动生成版本快照，支持查看历史版本并一键回滚到任意历史状态。

🔒

JWT 认证

完整的用户注册/登录体系，所有数据 API 通过 Bearer Token 鉴权保护。

🌐

已上线部署

通过 gunicorn + PythonAnywhere 部署到生产环境，支持 HTTPS 访问。

技术亮点

  • 连接池管理：MySQL 数据源使用 DBUtils PooledDB 连接池，maxconnections=6，线程安全复用
  • SQLite row_factory：使用 sqlite3.Row 实现字典游标，保证与 MySQL DictCursor 输出格式统一
  • HTTP API 自适应：自动识别 data/result/results/records/rows 等多种 JSON 包裹格式
  • 暗色主题：全站深色主题（#0a0f1d），ECharts dark 主题 + CSS backdrop-filter 毛玻璃效果
  • 响应式布局：CSS Grid 12 列网格 + GridStack.js 响应式列数调整
  • 应用工厂模式：Flask create_app() 工厂函数，支持多环境配置切换

演示看板
