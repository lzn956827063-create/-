项目概述

月相日记是一款面向女性用户的生理周期追踪微信小程序。核心设计理念是用 月相盈亏隐喻女性身体节律 — 新月代表经期开始，满月代表排卵期高峰，将科技与人文结合，让健康追踪不再冰冷。

项目完全使用微信原生框架开发（不使用第三方 UI 库），所有组件从零手写，包括 Canvas 月相绘制器、自定义星历日历组件、动画过渡效果等。

核心算法：自适应周期预测

predict(records, settings) → Prediction

Step 1 收集历史周期长度
Step 2 异常值过滤（丢弃 < 60% 或 > 140% 中位数的周期）
Step 3 滑动窗口 SMA（最近 2-6 个周期）
Step 4 加权移动平均趋势修正（±2 天）
Step 5 置信度计算（3 周期 → 0.3, 6+ → 0.825, 7+ → 1.0）
Step 6 构建日期→阶段映射表（覆盖 2 个周期）

功能特性

🌙

Canvas 月相绘制

纯手写 Canvas 2D 代码，根据当前周期阶段实时绘制月相图形（新月/弦月/满月），作为首页核心视觉元素。

📅

自定义星历日历

从零实现的日历组件，支持阴历阳历切换、标记生理周期日、预判排卵日与安全期，一目了然。

生物识别保护

集成 WeChat Soter 生物认证 API（指纹 + 面部识别），从后台切回前台时自动锁定，保护用户隐私。

每日日志记录

支持情绪、症状（腹痛/头痛/疲劳等）、体重、体温等多维度日常记录，数据本地持久化存储。

🔥

痛经舒缓工具包

内置穴位按摩指引、热敷计时器、舒缓音乐推荐等实用功能，帮助缓解经期不适。

📖

诗意文学内容

根据当前周期阶段展示匹配的文学段落 — 从古典诗词到现代散文，为产品注入人文温度。

技术架构

  • 纯原生开发：无任何第三方 UI 组件库，全部手写 WXML + WXSS + JS + Canvas
  • 模块化 utils 层：prediction.js（预测算法）、moon-renderer.js（Canvas 绘制）、storage.js（数据持久化）、cycle-calculator.js（阶段计算）
  • 自适应预测引擎：SMA 简单移动平均 + 异常值过滤（基于中位数 60%-140% 范围）+ WMA 加权趋势修正
  • custom-tab-bar：自定义底部导航栏，与月相主题统一视觉风格
  • 组件化设计：moon-phase / starry-calendar / record-popup / pain-toolkit / poetic-prose / period-history 六个独立组件
  • Canvas 2D API：实时渲染月相图形，支持动画过渡

开发背景

该项目源于对女性健康科技（FemTech）领域的关注。市面上多数经期追踪类产品要么功能繁杂广告多，要么 UI 设计缺乏美感。月相日记试图用极简的设计语言和精确的算法，打造一个"好用且美丽"的替代方案。

项目完全由个人独立完成，包括产品设计、算法实现、UI 绘制和全部前端代码。目前仍在开发迭代中，计划后续提交微信审核上线。
