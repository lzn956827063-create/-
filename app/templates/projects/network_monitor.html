🎓 毕业设计项目

企业内网实时监控系统

基于 Python Flask 框架与 SNMP 协议构建的企业级网络设备监控平台。 支持局域网设备自动发现、延迟/CPU/内存实时采集、异常检测告警、 数据可视化与 CSV 导出，适用于中小型企业 IT 运维场景。

🐍 Python 3 🌶️ Flask 📡 SNMP v2c 📊 Chart.js 🗄️ SQLite 🎨 Bootstrap 5 🔧 psutil 🧵 多线程

核心功能

🔍

设备自动发现

通过 ARP 表扫描 + ICMP Ping 自动发现局域网内所有活跃设备，支持自定义 IP 范围与扫描间隔。

📡

SNMP 交换机监控

通过 SNMP v2c 协议采集交换机端口流量、带宽利用率、包错误率等关键指标，支持多交换机并发监控。

📈

实时性能采集

定时轮询设备的延迟（ICMP RTT）、CPU 使用率与内存占用，历史数据缓存最多 120 个采样点供趋势分析。

⚠️

异常检测告警

可配置阈值（延迟/CPU/内存），超过阈值自动标记异常并记录告警历史。支持设备离线检测与通知。

📊

可视化面板

Chart.js 渲染实时趋势曲线与设备级历史图表。设备列表支持搜索、状态筛选、多维度排序。

💾

数据导出

支持按设备或全量导出 CSV 格式的历史监控数据，方便离线分析或导入 Excel/Navicat 进一步处理。

🏷️

设备别名管理

为每个 IP 设备设置别名与备注，支持手动添加/删除监控设备，不依赖自动扫描结果。

🖥️

设备详情抽屉

点击设备行弹出侧边抽屉，展示该设备的实时状态、离线次数、历史曲线、异常记录及元数据编辑。
